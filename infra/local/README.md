# Инфраструктура локального рабочего стенда

**Локальный рабочий стенд** — инстанс рабочего стенда, развернутый на вычислительной мощности разработчика, использующий виртуализацию или контейнеризацию для эмуляции зависимостей и сервисов, с ограниченной доступностью и упрощенной конфигурацией.

## Описание структуры папок

- `docker` - Docker-файлы.
- `nginx` - конфигурационные файлы веб-сервера.
- `pgadmin` - конфигурационные файлы PgAdmin.
- `settings.conf` - общий конфигурационный файл.
- `Makefile` - файл сборки локального рабочего стенда.
